# 🗺️ Nmap Tutor: The Interactive Modular Scanner

An interactive Bash wrapper for Nmap designed specifically for beginners and students. This script guides you through the process of building complex Nmap scans step-by-step, explaining the options along the way, and **reveals the exact Nmap command** it builds before executing it.

## 🤔 The Problem It Solves

When learning cybersecurity or network administration, Nmap can be intimidating. With hundreds of flags and options, beginners often fall into the trap of blindly copying and pasting commands from tutorials without actually understanding what `nmap -sS -p- -T4 -A -Pn` means.

**Nmap Tutor fixes this by:**
1. **Removing the guesswork:** It uses simple, conversational menus to ask you what you want to achieve (e.g., "Do you want to scan all ports or just the top 100?").
2. **Teaching through exposure:** After you make your selections, the script doesn't just run the scan in the background. It **prints the final command** to the screen. By seeing the command your choices created, you naturally memorize Nmap flags for future use. 
3. **Preventing syntax errors:** No more failed scans because of a misplaced dash or space. 

## ✨ Features

* **Interactive Menus:** Configure Targets, Ports, Scan Types, OS/Version Detection, Timing, Firewall Evasion, and Output Formats through numbered lists.
* **The "Command Reveal":** See the exact raw Nmap command generated by your choices before it runs.
* **Smart Privilege Escalation:** Automatically checks if your chosen scan requires root privileges (like `-sS` or `-O`) and prompts for `sudo` only if necessary.
* **Organized Output Management:** Easily save your scan results in Normal, Grepable, XML, or all formats.
* **Modular Design:** Easy to read and modify if you want to add your own custom Nmap flags later.

## 🛠️ How It Works (The Learning Process)

As you use the tool, you'll answer simple prompts:

> `[+] Select Scan Type:`
> `1) TCP SYN Stealth Scan [-sS]`
> `2) TCP Connect Scan [-sT]`
> `...`

Once you finish configuring your scan, the script pieces together your choices and presents the learning moment:

```text
=========================================================
Executing Command:
nmap -sS -p- -A -T4 -Pn -oN /home/user/results 192.168.1.50
=========================================================

```

Next time you need to do a fast, aggressive stealth scan ignoring ping, you won't need the script—you'll already know the flags!

* `nmap` must be installed (`sudo apt install nmap` on Debian/Ubuntu systems).
* Root/Sudo privileges (for advanced scan types).
